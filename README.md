# Devops journey-LBD
> "Mon parcours d'apprentissage Devops"- De Linux a kubernetes, en passant par le CI/CD,le cloud ,le monitoring ,la sécurité et le portfolio.

## Objectif
 Devenir **Devops engineer, en autodidacte, avec un portfolio solide defendable en entretien

**Philosophie** : *Learn By Doing* - Maitriser 15 outils à fond, pas en survoler plusieurs.
