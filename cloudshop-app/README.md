# 🛒 CloudShop — Le projet fil rouge

**CloudShop** est une application e-commerce simple qu'on fait évoluer pendant 10 mois. C'est le **cobaye DevOps** : on construit, conteneurise, déploie, surveille et sécurise étape par étape.

## 🎯 Évolution prévue

| Version | Mois | Capacité ajoutée |
|---------|------|------------------|
| v0.1 | M2 | Service Python basique |
| v0.5 | M3 | Conteneurisé avec Docker |
| v1.0 | M4 | Pipeline CI/CD |
| v1.5 | M5 | Infra Terraform |
| v2.0 | M6 | Multi-cloud (AWS + Azure) |
| v2.5 | M7 | Monitoring complet |
| v3.0 | M8 | Kubernetes |
| v3.5 | M9 | Sécurité (DevSecOps) |
| v4.0 | M10 | Production-ready + portfolio |

## 🛠️ Stack technique cible

- **Backend** : Python (FastAPI) ou Node.js
- **Database** : PostgreSQL
- **Container** : Docker
- **Orchestration** : Kubernetes
- **CI/CD** : GitHub Actions + Jenkins
- **Cloud** : AWS + Azure
- **Monitoring** : Prometheus + Grafana
- **Sécurité** : Vault, SonarQube, Trivy
