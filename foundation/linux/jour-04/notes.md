# 📅 Jour 4 — Users, Groups, Sudo

## 🎯 Commandes apprises

| Commande | Usage |
|----------|-------|
| `whoami` | Affiche l'user courant |
| `id` | Affiche UID, GID et groupes |
| `groups` | Liste les groupes d'un user |
| `useradd` | Crée un user |
| `userdel` | Supprime un user |
| `chown` | Change le propriétaire |
| `sudo` | Exécute en tant que root |
| `cat /etc/passwd` | Liste tous les users |
| `cat /etc/group` | Liste tous les groupes |
